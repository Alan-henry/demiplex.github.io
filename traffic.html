A.I based traffic control system

We use an automated rtaffic light system which is quite common but thats not the star of the show, we have implimented a motion detector system on the streets which can detect if somoene has run a red light or is over the speed limit by calculating its velocity using the frames per seconds it takes for that vehical to reach from one point to another, after which the security camera takes a picture of the lisence plate of the vehical and then is called for a court hearing and necessary actions are taken against the individual.
